## Qué es esto

Este pack aterriza la empresa (modelo de negocio + operación + finanzas + cumplimiento) para un negocio de **Import/Export & Sourcing B2B**.

Tu “producto real” es reducir incertidumbre del comprador y del proveedor mediante:

1. Precio final (landed cost) verificable
2. Calidad (especificación + inspección + evidencias)
3. Cumplimiento (documentos, normas, HS code, etiquetado/permisos)
4. Entrega (Incoterm correcto + logística + aduana + última milla si aplica)

## Cómo usarlo (mínimo viable, profesional)

1. Completa en el documento maestro la sección "Parámetros del corredor" (origen/destino/capital/categorías).
2. Usa la plantilla RFQ para pedir cotizaciones comparables (mismo Incoterm y especificación).
3. Construye cotizaciones con costo puesto usando la plantilla de landed cost.
4. Opera con checklist documental (antes/durante/después del embarque).
5. Formaliza con términos (cliente/proveedor) y ejecuta el plan 0–90 días.
