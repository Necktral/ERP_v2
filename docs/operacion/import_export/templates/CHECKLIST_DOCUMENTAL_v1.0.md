## 1) Antes de producción (pre-PO)

- [ ] Especificación cerrada (ficha técnica aprobada)
- [ ] Precio, MOQ y tiempos confirmados
- [ ] Términos de pago acordados
- [ ] Compliance básico (producto/país/restricciones)

## 2) Antes de embarque (pre-shipment)

- [ ] Inspección (si aplica): checklist + evidencia (fotos/video)
- [ ] Factura comercial (datos completos)
- [ ] Packing list (pesos/volúmenes consistentes)
- [ ] Etiquetado conforme (idioma/lote/origen/adverts)
- [ ] Certificados específicos (si aplica)

## 3) Embarque

- [ ] BL/AWB emitido y verificado
- [ ] Seguro emitido (si aplica)
- [ ] Documentos enviados a agente aduanal

## 4) Aduana / nacionalización

- [ ] HS code revisado por agente aduanal
- [ ] Declaración consistente (valor, descripción, cantidades)
- [ ] Permisos/avisos previos (si aplica)
- [ ] Pago de aranceles/impuestos confirmado

## 5) Liberación y entrega

- [ ] Liberación aduanal
- [ ] Transporte interno coordinado
- [ ] Evidencia de entrega (POD)

## 6) Postventa y cierre

- [ ] Costos reales vs estimados (landed cost real)
- [ ] Incidencias (daños, demoras, faltantes)
- [ ] Recomendaciones de ajuste (buffer, proveedor, Incoterm)
- [ ] Reorden: propuesta de calendario y abastecimiento
