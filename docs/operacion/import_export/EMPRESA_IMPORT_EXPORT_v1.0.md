## 0) Resumen ejecutivo (1 página)

**Tipo de empresa:** Trading Company / Import-Export & Sourcing (B2B)

**Propuesta de valor:** ejecutar compras internacionales como una transacción controlada, con precio final verificable, calidad, cumplimiento y entrega.

**Qué vendemos realmente:** reducción de incertidumbre y fricción (riesgo → control → evidencia).

**Qué compra el cliente:**

- Precio final (landed cost) con reglas claras
- Calidad verificable (especificación + inspección + evidencia)
- Cumplimiento (documentación, HS code correcto, etiquetado/permisos)
- Calendario y visibilidad (tracking + hitos)

## 2) Modelos de ingresos (A–D) y recomendación de arranque

### Modelo A — Comisión de intermediación (Broker / Facilitator)

- Cobro: 3%–10% por operación (según complejidad y riesgo)
- No compras inventario
- Ventaja: baja presión de caja
- Riesgo: desintermediación (que te “salten”)

**Blindaje mínimo (operativo):**

- Contrato de exclusividad por SKU/territorio/tiempo
- Fee por reorden
- Tú controlas la documentación + inspección + coordinación

### Modelo B — Sourcing + gestión integral (retainer + success fee)

- Cobro: retainer mensual + fee por orden
- Ideal: clientes recurrentes
- Ventaja: ingresos estables

### Modelo C — Importador/Exportador de registro con margen (Reseller)

- Tú compras y revendes
- Margen: 15%–40% según rubro
- Ventaja: control total del cliente
- Riesgo: capital + inventario + demanda

### Modelo D — Marca propia (Private Label)

- Alto potencial, alta exigencia (calidad, registros, empaque, marketing, caja)
- Recomendación: solo cuando A/B/C están dominados con datos reales

### Recomendación (robusta, sin “plan mínimo”)

- Arranque: **A + B** para flujo y cartera
- Paralelo: preparas capacidad legal/operativa para ejecutar **C** en 1–2 categorías de alta rotación
- Escalado: **D** cuando haya datos y control de supply/quality

## 3) Selección de productos (filtro brutal: margen vs riesgo)

Usar scoring 1–5 por criterio. Objetivo: puntajes altos y riesgos controlables.

1. Margen y estructura de costos

- ¿Margen bruto realista ≥ 20% incluyendo flete/seguro/aduana?
- ¿Flete penaliza por volumen/peso?
- ¿Sensibilidad al precio (commodity vs especializado)?

2. Riesgo regulatorio/documental

- ¿Requiere permisos sanitarios, eléctricos, juguetes, químicos, etc.?
- ¿Etiquetado/manuales/idioma obligatorios?
- ¿Riesgo de confiscación por HS code o marca?

3. Riesgo de calidad/devoluciones

- ¿Se inspecciona con checklist objetivo?
- ¿Estándares claros (medidas, tolerancias, materiales)?

4. Liquidez comercial

- ¿Compra recurrente B2B?
- ¿Se puede vender por abastecimiento mensual?

5. Riesgo financiero (cobro)

- ¿Cliente paga contado/anticipo?
- ¿Pago escalonado/LC viable?

**Categorías normalmente más amigables (depende del país):**

- Consumibles B2B, repuestos estandarizados, empaques, ferretería seleccionada, accesorios industriales simples.

**Categorías normalmente de alto riesgo al inicio:**

- Alimentos, cosméticos, médicos/suplementos, baterías/energía regulada, químicos, juguetes regulados, electrónicos con homologación compleja.

## 4) Pipeline operativo (sin improvisación)

### Paso 1 — Onboarding del cliente

- Ficha cliente: razón social, dirección fiscal, responsable, capacidad de pago
- Ficha técnica del producto: especificación, tolerancias, empaque
- Objetivo: volumen mensual, precio objetivo, plazo

**Output:** brief + especificación cerrada.

### Paso 2 — Sourcing y validación de proveedor

- 3 cotizaciones comparables (mismo Incoterm, misma especificación)
- Verificación mínima: existencia legal + referencias + evidencia (foto/video) + muestras si aplica
- Inspección pre-embarque (tercero) si el monto lo amerita

**Output:** shortlist (3 proveedores) + evidencia.

### Paso 3 — Cotización correcta (costo puesto)

Incluye:

- Precio EXW/FOB/etc. (Incoterm)
- Flete internacional
- Seguro
- Aduana (aranceles + IVA/impuestos + agencia)
- Transporte interno destino
- Costo financiero (tiempo del dinero)
- Tu fee/margen
- Buffer de riesgo explícito

**Output:** cotización landed cost versionada + validez.

### Paso 4 — Contrato + Incoterm + pagos

- Incoterm coherente con control real
- Pagos internacionales: T/T por hitos, LC si aplica, cobranza documentaria

**Output:** términos firmados + hitos de pago.

### Paso 5 — Documentación (checklist)

- Factura comercial
- Packing list
- BL/AWB
- Certificado de origen (si aplica)
- Certificados específicos (MSDS, conformidad, sanitario, etc.)

**Output:** carpeta documental completa + verificación previa al embarque.

### Paso 6 — Aduana y liberación

- HS code correcto
- Declaración consistente (precio, descripción, cantidades)
- Gestión de inspecciones

**Output:** liberación + trazabilidad de costos reales.

### Paso 7 — Postventa y repetición

- Evidencias (fotos/inspección), trazabilidad básica
- Reorden y abastecimiento

**Output:** reorden programado.

## 5) Riesgos que quiebran importadoras (y mitigación)

1. Cotización falsa (no incluía todo)

- Mitigación: checklist + landed cost + buffer + validación con forwarder/agente aduanal antes de prometer precio final.

2. Calidad (llega y no sirve)

- Mitigación: especificación + muestra + inspección pre-embarque + cláusula de reclamación.

3. Estafa/proveedor fantasma

- Mitigación: verificación escalonada + pagos por hitos + evitar 100% adelantado salvo proveedor top.

4. Aduana (retención/multas)

- Mitigación: HS code correcto, documentación impecable, cumplimiento del rubro.

5. Tipo de cambio

- Mitigación: validez corta + cláusula FX + cobro anticipado + coberturas si volumen lo justifica.

## 7) Estructura legal/fiscal (marco por capas)

Sin país no se puede cerrar al 100%, pero sí el marco:

- Capa A: entidad legal + cuenta bancaria + actividad habilitada
- Capa B: habilitación aduanera (importador/exportador) y agente aduanal
- Capa C: contratos (cliente/proveedor) + límites de responsabilidad
- Capa D: compliance (screening básico por sanciones/restricciones)

**Nota:** validar con asesoría legal/fiscal local.

## 8) Plan de ejecución 0–90 días (medible)

### Días 0–10: base operativa

1. Definir 1 corredor + 1–2 categorías
2. Preparar plantillas (RFQ, cotización landed cost, checklist documental, términos)
3. Conseguir 2 aliados: forwarder + agente aduanal

**Entregable:** kit de operación (documentos + checklist + flujo).

### Días 11–30: sourcing real

1. Lista de 30 proveedores
2. Filtrar a 8–10, cotizaciones comparables
3. Seleccionar 3 finalistas
4. Preparar 1 oferta comercial por segmento

**Entregable:** 3 líneas de suministro listas (precio/tiempos/documentos).

### Días 31–60: ventas y órdenes piloto

1. Lista de 150 prospectos B2B
2. Cadencia de contacto
3. 3–5 reuniones
4. 5–10 cotizaciones formales
5. Negociar términos de pago por hitos

**Entregable:** 1–2 operaciones piloto firmadas.

### Días 61–90: operación + repetición

1. Ejecutar piloto con control documental total
2. Medir: tiempos, costos reales vs estimados, incidencias
3. Ajustar pricing, buffer y checklist
4. Convertir piloto en reorden/abastecimiento

**Entregable:** 1 cliente recurrente o 2 con reorden programado.
